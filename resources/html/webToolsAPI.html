Endpoint Parameters Description
/convertTimeFromEpoch data Convert Readable Time to Epoch Time
/convertTimeToEpoch data Convert Epoch Time to Readable Time
/validateJson data, pretty, whiteListFilter, blackListFilter Validate a JSON object. Optional filter parameters allow for white or black list filering of the response. The repsonse is also formatted if pretty is set to true. NB: If both fitlers are present whitelist filter only will be applied
/copareJson jsonOne, jsonTwo Compare two JSON objects. A JSON object containing only the differences is returned.
/base64Encode data Base 64 encode a string.
/base64Decode data Base 64 decode an encoded string.
/urlEncode encode,decode URL encode (or decode) a string based on the query parameter.
/hashing data, hashType Create an MD5 or SHA-1 (based on the hashType param) Hash of a string.
